# **App Name**: PraiseMotion Hub

## Core Features:

- Homepage Hero: Rotating hero banner showcasing latest music, events, and donation prompts.
- Music & Media Gallery: Filterable gallery for songs, videos, and live sessions with embedded audio/video players (Spotify, Apple Music, YouTube).
- Events Calendar: Display upcoming concerts, worship nights and livestreams with RSVP/booking functionality.
- Secure Donation System: Accept donations via Mpesa, PayPal, Stripe, and credit/debit cards, with recurring donation options and automated acknowledgements.
- Social Media Integration: Automated feeds from Instagram, TikTok, and YouTube.
- AI-Powered Content Suggestions: Generative AI tool that provides devotional topics based on recent blog posts and trending musical themes.
- Contact and Location: Contact form, links to social media, and Google Maps integration for location.
